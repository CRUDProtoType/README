# Basic CRUD Project
==================
## 기본규칙
- 3가지 버전 작성
- FullStack 
- Backend는 API만 제공
- FrontEnd는 API에 맞춘 데이터로만 작성
- Continer 기분 가동
- OS : RockyLinux 9 
- 기본 기능
   - 회원가입
   - 로그인
   - 글 작성
   - 글 리스트
   - 글 수정
   - 글 삭제
